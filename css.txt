How to include css in html

3 ways / method

1. Inline -> tag
2. Internal -> in html
3. External -> .css


CSS Selectors - CSS selectors are used to find (or select) the HTML elements you want to style.

- Universal 
- ID
- Class
- Tag
- Combinator
    - Descendant (space): Matches all elements that are descendants (children, grandchildren, etc.) 
                          of a specified element
    - Child (>): Selects all elements that are direct children of a specified element.
    - Next Sibling (+): Select an element that is directly after a specific element.
    - Subsequent-sibling (~): Selects all elements that are next siblings of a specified element.
- Group

Font properties: size, style, weight, family
Text properties: color, align, spacing
Box Model - border, padding, margin, content
Background properties: color, image, repeat, position
Borders: width, style, radius,color,border sides, border shorthand

Box Model: The CSS Box Model is a layout model that represents every HTML element as a box containing content, padding, border, and margin.

Content: Text/image/anything
Padding: space between content and border
border: outer layer of padding and content, act as an edge of content.
margin: space outside the border.

z-index controls which element appears on top when elements overlap
Controls what happens when content is too large for its container.
Controls transparency of an element.
Variables store reusable values. Also called custom properties.

psudo-class: target an element's state
psudo-elements: target specific parts of an element or create virtual content
